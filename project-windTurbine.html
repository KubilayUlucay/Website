Wind Turbine Grid Integration Analysis

Analyzing the impact and cost-effectiveness of wiring configurations for wind turbine grid integration using PowerWorld Simulator.

Project Overview

This project was undertaken for a componay, a leading wind energy developer, to plan the optimal and lowest-costing design for integrating a new 200 MW wind farm into the existing Metropolis power grid. The scope involved evaluating various connection scenarios for Type 3 DFIG turbines, considering factors like substation interconnection points, conductor types, system losses, and overall cost-effectiveness over a 5-year period.

The core of the project was to perform detailed simulations and analyses using PowerWorld Simulator to determine the most reliable and economically viable configuration for feeding the wind farm's output into two separate interconnection substations, adhering to company's technical and traditional requirements.

Key Features & Technologies

  • Power System Simulation: Utilized PowerWorld Simulator for detailed modeling and analysis of the power grid and various wind farm integration scenarios.
  • Cost-Benefit Analysis: Evaluated different transmission line configurations and conductor types based on initial costs and the cost of system losses over 5 years.
  • System Loss Minimization: Investigated configurations to minimize power losses in the transmission system while integrating the new 200 MW wind farm.
  • Contingency & Reliability Analysis: Considered reliability by requiring connection to two separate substations and ensuring conductor capacities were not exceeded.
  • Technical Reporting: Documented 27 different experiments, their outcomes, and provided a final budgetary calculation and recommendation.
  • Technologies Used:
    • Software: PowerWorld Simulator
    • Analysis Techniques: Load flow analysis, economic dispatch considerations (implied by cost analysis), transmission line parameter calculations.

Outcomes & Learning

The analysis concluded that connecting the companys' wind farm to the PETE and PAI substations using CONDOR type conductors (Experiment 9) presented the most economically viable and technically sound solution. This configuration offered the best balance between initial conductor costs and long-term system losses, resulting in the lowest overall cost over a 5-year scope while ensuring system stability and reliability.

This project provided significant hands-on experience in power system analysis using industry-standard software. Key learnings included the complex interplay between conductor characteristics, transmission distances, system loading, and overall economic impact. It also reinforced the importance of meticulous data collection, systematic experimentation, and clear documentation in power engineering projects.
